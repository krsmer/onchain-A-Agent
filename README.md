# Onchain AI Agent

TypeScript tabanlı onchain AI agent projesi.

## Kurulum

Önce bağımlılıkları yükleyin:

```bash
npm install
```

## Kullanım

### Geliştirme Modu

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Çalıştırma

```bash
npm start
```

## Proje Yapısı

```
onchain-AıAgent/
├── src/           # TypeScript kaynak dosyaları
├── dist/          # Derlenmiş JavaScript dosyaları
├── package.json   # Proje bağımlılıkları
└── tsconfig.json  # TypeScript yapılandırması
```
